Zadanie rekrutacyjne
Program uruchamia się w IDE obsługującym język Java przy pomocy jdk 1.8.
Uruchamiamy klasę Calculator, w której znajduje się metoda main.
Jako argument można podać ścieżkę do pliku xml z kursami walut. W przypadku braku argumentów uruchomienia program wczyta plik domyślny.
Następnie komunikacja z programem ma miejsce w wierszu poleceń:
- kiedy program spyta o wartość w euro, należy podać liczbę nieujemną
- kiedy program spyta o walutę należy wpisać jeden ze skrótów odnoczący się do konkretnej waluty (np PLN)
